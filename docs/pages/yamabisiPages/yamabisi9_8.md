# Yamabisi 9.8 hp Four Stroke Outboard

## Yamabisi 9.8hp Stats:
![Yamabisi 9.8hp stats](../../images/yamabisiImages/yamabisi9_8_stats.jpg "Yamabisi 9.8hp stats")

## Clone of Tohatsu MFS9.8

## Oil:
- 0.8l 10w-40 marine oil

## Tohatsu Parts:

### Water Pump Kit:
- Must reuse Yamabisi key, key from rebuild kit 1-2mm too long (can be cut)
- Nissan/Tohatsu Part Number - 3B287322M
- Mercury Part Number - 803750A03

### Fuel Pump:
- Requires two M6-1.00x25mm bolts
- Nissan/Tohatsu Part Number - 3H604000
- Mercury Part Number - 803529T06

### Ignition Coil:
- Nissan/Tohatsu Part Number - 3V1060400M

### Thermostat:
- 3H6-01030-0 <-- Not sure this is correct?
- THERMOSTAT (71 DEGREES CELCIUS) 3AA010300

### Carb Rebuild Kit:
- 3V1-87122-1M <-- Not sure this is correct? 

### Fuel Line:
- 3H670-2002M -- verify?
- Yamabisi used Yamaha style connector on stock fuel tank

### Carb Jets:
- For standard sea-level: main jet #92 (3V2-04492-0), slow jet #40 (3V1-04540-0).
- For ~6500 ft: main #90 (3V2-04490-0), slow #38 (3V1-04538-0).
- For 6600-9800 ft: main #87 (3V2-04487-0), slow #37 (3V1-04537-0).
